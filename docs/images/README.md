# Documentation Images

This directory contains visual documentation images used in the technical reports.

## Images

### 1. `pose_landmark_topology.svg`
**Description:** MediaPipe pose landmark topology diagram showing the 33 anatomical landmarks and their skeletal connections.

**Usage:** Referenced in both Technical_Report.md and Detailed_Technical_Report.md to illustrate the MediaPipe pose detection structure.

**Content:**
- Complete skeletal structure of MediaPipe's 33 landmarks
- Connections between landmarks showing body topology
- Visual reference for understanding landmark indices and relationships

---

### 2. `landmarked_pose.jpeg`
**Description:** Real-world example of MediaPipe pose detection applied to an actual person with all 33 landmarks overlaid.

**Usage:** Demonstrates the practical application of pose detection in the body scanning application.

**Content:**
- Subject with MediaPipe landmarks visible
- Shows detection accuracy across all body regions
- Validates system performance on real-world images

---

### 3. `sample_measurement_on_ui.jpeg`
**Description:** Screenshot of the application UI displaying sample anthropometric measurements with measurement names, values, and confidence scores.

**Usage:** Illustrates how measurement results are presented to users in the application interface.

**Content:**
- Measurement name labels
- Calculated values in centimeters
- Confidence scores for each measurement
- User-friendly presentation format

---

## Image Specifications

- **Format:** SVG (vector) for topology diagram, JPEG for photographs/screenshots
- **Purpose:** Visual documentation to support technical reports
- **License:** Project documentation - see main project license
